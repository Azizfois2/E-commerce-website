## 🔧 Customization

### Change Colors
Edit in `cinematic-enhancements.css`:
```css
:root {
  --cyan: #00f5d4;  /* Change this */
  --orange: #ff6b35; /* And this */
}
```

### Disable Custom Cursor
Remove from `index.html`:
```html
<!-- Remove this line -->
<script src="assets/js/cinematic-enhancements.js"></script>
```

### Adjust Animation Speed
In `cinematic-enhancements.css`:
```css
* {
  transition-duration: 0.3s; /* Change this */
}
```

## 🐛 Troubleshooting

**Cursor not showing?**
- Only works on desktop (>1024px width)
- Check browser console for errors

**Animations laggy?**
- Reduce motion in OS settings
- Close other browser tabs
- Update graphics drivers

**Search not working?**
- Ensure `data.js` is loaded
- Check browser console
- Verify products array exists

**Page transitions too slow?**
- Edit `page-transitions.js`
- Reduce timeout values
